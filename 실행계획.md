## 3. Phase 0: PoC/기술 검증 (2-3일)

### 3.0 목표
> **"핵심 기술이 실제로 동작하는지 검증"**

MVP 개발 전에 반드시 확인해야 할 기술적 리스크:
1. Firecrawl이 쿠팡을 스크래핑할 수 있는가?
2. LLM이 데이터를 잘 추출하는가?
3. LLM 인사이트 품질이 충분한가?

### 3.1 태스크 목록

```
Phase 0: PoC (예상 2-3일)
│
├── Day 1: 환경 설정 & Firecrawl 테스트
│   ├── [ ] 0.1.1 Python 가상환경 설정
│   ├── [ ] 0.1.2 Firecrawl 로컬 설치 (Docker)
│   ├── [ ] 0.1.3 Firecrawl API 테스트 (간단한 사이트)
│   ├── [ ] 0.1.4 쿠팡 카테고리 페이지 스크래핑 테스트
│   └── [ ] 0.1.5 결과물: 쿠팡 HTML/Markdown 데이터 확보
│
├── Day 2: LLM 파이프라인 테스트
│   ├── [ ] 0.2.1 OpenRouter API 키 발급
```

| 항목 | 기준 | 통과 |
|------|------|------|
